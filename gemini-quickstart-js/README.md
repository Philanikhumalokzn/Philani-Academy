# Gemini API quickstart (JavaScript)

This folder is a standalone test app that follows the official Gemini API JavaScript quickstart.

## Prereqs
- Node.js installed
- A Gemini API key from Google AI Studio

## Run (PowerShell)
```powershell
cd gemini-quickstart-js
$env:GEMINI_API_KEY = "YOUR_KEY_HERE"
npm install
npm start
```

## Docs used
- Quickstart (JavaScript): https://ai.google.dev/gemini-api/docs/quickstart#javascript_1
- API keys: https://ai.google.dev/gemini-api/docs/api-key
- Libraries: https://ai.google.dev/gemini-api/docs/libraries
- generateContent API: https://ai.google.dev/api/generate-content#method:-models.generatecontent
